### Toán tử(Operator)
-[Toán tử 1 ngôi](#toan-tu-1-ngoi)

| ++  | tăng lên 1 đơn vị | x++    |
| --- | ------------------ | ------ |
| --  | giảm đi một đơn vị | y--    |
| =   | x = 5              | x = 5  |
| +=  | x += 3             | x = x + 3 |
| -=  | x -= 3             | x = x - 3 |
| *=  | x *= 3             | x = x * 3 |
| /=  | x /= 3             | x = x / 3 |
| %   | x %= 3             | x = x % 3 |

-[Toán tử 2 ngôi](#toan-tu-2-ngoi)

| toán tử | mô tả       | Ví dụ  |
| ------- | ----------- | ------ |
| +       | cộng        | x + y  |
| -       | trừ         | x - y  |
| *       | nhân        | x * y  |
| **      | lũy thừa    | x ** y |
| /       | chia        | x / y  |
| %       | chia lấy dư | x % y  |

VD: number x  {26}

number y  {33}

number z  {x + y}

number t { z - 16}

out(z) ~~59

out(t) ~~43

- [Toán tử so sánh](#toan-tu-so-sanh)

| Toán tử | Mô tả                              |
| ------- | ---------------------------------- |
| ==      | bằng                               |
| ===     | bằng giá trị và kiểu dữ liệu      |
| !=      | không bằng                         |
| !==     | không bằng giá trị và kiểu dữ liệu |
| >       | Lớn hơn                            |
| <       | Nhỏ hơn                            |
| >=      | Lớn hơn bằng                       |
| <=      | Nhỏ hơn bằng                       |

VD:

number x {5}

number y {10}

out(x > y)//false

out(x == y)//false

out(x === y)//false

out(x != y)//true

out(x < y)//true
